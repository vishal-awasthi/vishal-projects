********************************Manage Numbers application******************************************

This application is a rest micro service which is IntelliJ project. It can be directly imported on IntelliJ IDE. It is made using spring boot so it could run independently. 
The main functionality is present in Service class: ManageNumbersService. It uses quick sort to sort the values. 

The working of the application can be tested either by running following REST API: 
http://localhost:9091/web-back-end/manageNumbers/sortAndLimit 
Or by running the Junit file of ManageNumbersService. 
In both the above cases, the input file is summary.csv and output is summary_result.csv
